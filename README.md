## ⚙️ Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/EERLA-VENKATESH/Finance-Dashboard-UI.git
```

### 2️⃣ Navigate to Project

```bash
cd Finance-Dashboard-UI/finance-dashboard
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Run Application

```bash
npm start
```
